# Security

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it by emailing **sachin@example.com**.

Do **not** create a public GitHub issue for security vulnerabilities.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Disclosure Policy

- Reports are acknowledged within 48 hours.
- We aim to triage within 5 business days.
- Fixes are released as patch versions.
- Public disclosure is coordinated after a fix is available.
